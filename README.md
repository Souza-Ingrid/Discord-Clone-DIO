# 🎧 Clone da Página Inicial do Discord

Projeto desenvolvido como atividade prática da **DIO (Digital Innovation One)** com o objetivo de recriar a interface da página inicial do Discord utilizando apenas HTML e CSS.

---

## 📌 Objetivo do Projeto

O principal objetivo deste projeto foi:

* Praticar **HTML5 semântico**
* Aplicar **CSS3 com organização profissional**
* Utilizar **Flexbox para alinhamentos**
* Trabalhar **responsividade com Media Queries**
* Desenvolver organização de código com comentários explicativos
* Reproduzir fielmente um layout moderno

Este projeto simula a página inicial do Discord, focando na estrutura visual e organização dos elementos.

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Flexbox
* Media Queries (Responsividade)

## 📱 Responsividade

O projeto foi desenvolvido com foco em adaptação para dispositivos móveis.

Foi utilizada a seguinte media query:

```css
@media (max-width: 768px)
```

No mobile:

* Elementos passam a ficar empilhados
* Botões ocupam largura total
* Títulos reduzem tamanho
* Layout se adapta à tela menor

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, foram praticados:

* Reset CSS
* Organização por blocos comentados
* Uso de `box-sizing`
* Alinhamento com `display: flex`
* Uso de `justify-content` e `align-items`
* Controle de largura com `max-width`
* Pseudo-classes como `:hover`
* Seletores como `:first-child`, `:last-child` e `:nth-child`
* Estruturação limpa e escalável
